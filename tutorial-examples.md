# 인공지능 확장 활용 아이디어

## 소개 @unplugged

인공지능 확장의 세 가지 핵심 기능을 어떻게 활용할 수 있는지 알아봅시다!

## 엔티티 감지 활용 @unplugged

**엔티티 감지**는 에이전트나 플레이어 주변의 몹과 플레이어를 찾아냅니다.

활용 아이디어:
- **몬스터 경보 시스템**: 주변에 몬스터가 나타나면 자동으로 경고 메시지 전송
- **플레이어 탐색**: 주변 플레이어를 찾아서 안내 메시지 전달
- **동물 감지**: 특정 몹이 근처에 있을 때만 동작하는 자동화 프로그램

감지 기준(에이전트/플레이어)과 대상(전체/플레이어/몬스터/몹), 반경을 조합하면 다양한 상황에 맞는 감지 시스템을 만들 수 있습니다.

## 블록 데이터 분석 활용 @unplugged

**블록 데이터 분석**은 에이전트가 지나간 길의 블록 구성을 파악합니다.

활용 아이디어:
- **지형 조사**: 잔디·흙·돌 비율을 분석하여 땅의 특성 파악
- **광물 탐색**: 특정 광석 블록을 추적하여 채굴 효율 측정
- **환경 모니터링**: 여러 방향으로 블록을 분석하여 주변 환경 데이터 수집

초기화 → 추적 블록 등록 → 반복 분석 → 결과 출력 순서로 사용하면 체계적인 블록 데이터를 얻을 수 있습니다.

## 에이전트 인벤토리 활용 @unplugged

**에이전트 인벤토리**는 채굴하면서 쌓인 아이템을 자동으로 관리합니다.

활용 아이디어:
- **자동 정리**: 채굴 후 섞인 아이템을 종류별로 자동 정렬
- **선별 버리기**: 불필요한 블록만 골라서 지정 방향에 내려놓기
- **분류 시스템**: 분석 → 정렬 → 필요 없는 블록 버리기를 하나의 명령으로 자동화

세 가지 기능을 조합하면 감지 → 채굴 → 분석 → 인벤토리 관리까지 이어지는 완전 자동화 프로그램을 만들 수 있습니다!
